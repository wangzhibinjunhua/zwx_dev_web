Huayinghealth API


  • get_bpbg_v1.php

    获取血压计和血糖仪数据

    • 参数说明
      • devicetype:设备型号 :必须
        • 1:血压计
        • 2:血糖仪
      • sn:设备序列号 :必须
      • user:血压计用户编号 :可选
        • 0:用户0
        • 1:用户1
        • 空缺:所有用户
      • time:时间 :可选 :查询此时间之后的数据 :格式 Y-m-d H:i:s
      • count:数据条数 :可选 :查询最新count数量的数据,默认查询最新一条
    • 返回值
      • JSON数据格式
    • 返回值实例
      • 例子:http://api.huayinghealth.com/gsm/bpbg/get_bpbg_v1.php?devicetype=1&sn=42949834773
        • 返回: {"code":1000,"message":[{"id":"21","sn":"42949834773","imsi":"3152546943448239","user":"1","bp":"132*092","bpm":"78","datetime":"2016-06-27 19:29:01"}]}
          • bp:血压,高压*低压 格式
          • bpm:心率
      • 例子:http://api.huayinghealth.com/gsm/bpbg/get_bpbg_v1.php?devicetype=2&sn=352321538
        • 返回: {"code":1000,"message":[{"id":"43","sn":"352321538","imsi":"3152563827009178","bg":"105","datetime":"2016-06-28 15:26:44"}]}
          • bg: 血糖 mg/dL
    • 错误代码
      • 1000:获取数据成功.
      • 1008:无符合条件的查询数据.
      • 1005:user(用户)无效.
      • 1006:设备类型不支持.
      • 1007:sn为空.
